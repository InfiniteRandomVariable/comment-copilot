# SKILL.md Compiler Spec

A single canonical `SKILL.md` exists per account version.

Generation inputs:
- Owner-selected persona profile.
- Title/caption pattern signals.
- Historical interaction signals.

Required sections:
1. Agent Identity
2. Brand Voice
3. Expertise Scope
4. Comment Intent Playbook
5. Safety Guardrails
6. Escalation Rules
7. Example Responses
8. Prohibited Content

Approval policy:
- Every new version starts as `draft`.
- Only explicit owner approval activates a version.
- Active version is bound to every generated reply candidate.
